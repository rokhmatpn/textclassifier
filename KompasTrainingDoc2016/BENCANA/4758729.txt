	Tekanan Udara yang Lebih Rendah Geser Penyebaran Asap ke Sumbar


	 
	Padang, Kompas - Kabut asap akibat kebakaran hutan di Provinsi Riau tak hanya mengarah ke Jambi, tetapi juga bergeser ke wilayah Sumatera Barat. Hal itu karena tekanan udara di bagian selatan Pulau Sumatera jauh lebih rendah dibandingkan tekanan udara di bagian utara pulau tersebut.
	Prakirawan Badan Meteorologi, Klimatologi, dan Geofisika (BMKG) Bandara Internasional Minangkabau, Siska Anggreini, di Padang, Selasa (11/2), mengatakan, meski relatif masih aman, masyarakat tetap diminta untuk mewaspadai kondisi tersebut. Agar tidak memperparah kabut asap, masyarakat diharapkan juga tidak bertindak gegabah yang bisa menyebabkan titik api baru di Sumatera Barat (Sumbar).
	Berdasarkan citra satelit, memang masih terdeteksi titik api di sejumlah wilayah, seperti Aceh, Sumatera Utara (Sumut), dan Riau. Namun, dengan tekanan udara yang lebih tinggi di bagian utara, pergerakan asap dari 36 titik api di Riau semakin cepat ke selatan, termasuk ke Sumbar, papar Siska.
	Menurut dia, wilayah di Sumbar yang kini terdampak kabut asap berbatasan dengan Provinsi Riau, seperti Kabupaten 50 Kota, Kota Payakumbuh, Kabupaten Pasaman Timur, Kota Sawahlunto, dan Sijunjung. Sumbar masih terdampak, belum terdeteksi adanya titik api. Jarak pandang juga masih 5-8 kilometer sehingga masih aman, tutur Siska.
	Kepala Seksi Observasi dan Informasi BMKG Bandara Internasional Minangkabau Budi Samiadji menambahkan, meski saat ini dampak kabut asap relatif aman untuk kesehatan ataupun penerbangan, masyarakat diminta waspada. Mengacu pada pengalaman tahun sebelumnya, kabut asap tak pernah mengganggu penerbangan. Jarak pandang terendah hanya 2 kilometer, kata Budi.
	BMKG memperkirakan kabut asap masih akan terus berlangsung hingga akhir Maret, seiring potensi hujan yang diprediksikan pada April-Mei mendatang.
	Adapun berdasarkan pantauan Satelit NOAA-18, jumlah titik api semakin bertambah. Jika pekan lalu titik yang terpantau maksimal 20 titik, pada Selasa ini Satelit NOAA-18 memantau 73 titik api. Senin sebelumnya, baru terdapat 55 titik api di Sumut.
	NOAA-18 juga melaporkan kenaikan jumlah titik api di Pulau Sumatera dari 187 titik pada hari Senin menjadi 458 titik pada Selasa ini. Kepala Bidang Data dan Informasi Balai Besar Meteorologi, Klimatologi, dan Geofisika Wilayah I Medan Hendra Suwarta mengatakan, saat ini Sumut memang memasuki puncak musim kemarau sehingga berpotensi menaikkan jumlah titik api. Kami belum tahu apakah titik api itu karena pembakaran hutan atau cuaca panas, tutur Hendra.
	Picu kebakaran
	Sementara itu, BMKG Aceh memprediksi, musim kemarau akan datang lebih cepat di Aceh. Kondisi ini memicu semakin tingginya risiko kebakaran hutan di wilayah tersebut.
	Kami imbau masyarakat tak menyulut api, seperti membuka lahan dengan membakar ataupun membuang puntung rokok sembarangan. Sebab, di lahan-lahan kering, api kecil cepat meluas, tutur Khairul Akbar, Koordinator Analis Badan Meteorologi, Klimatologi, dan Geofisika Aceh.
