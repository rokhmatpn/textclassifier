
	Kabut Asap
	Kebakaran Lahan Terus Terjadi


	PONTIANAK, KOMPAS - Kebakaran lahan yang menimbulkan kabut asap di sejumlah wilayah masih terus terjadi. Pada Rabu (19/2) di Kalimantan Barat terpantau ada 55 titik api, sementara pada Selasa di Kalimantan Tengah terdapat 17 titik api. Umumnya, titik api tersebut akibat pembakaran lahan untuk perkebunan oleh masyarakat.
	Titik api di Kalbar terdapat di Kabupaten Ketapang sebanyak 32 titik, Kabupaten Pontianak 9 titik, dan Kubu Raya 8 titik. Titik api masih bermunculan meskipun cuaca di Kalbar beberapa hari ini relatif berawan, bahkan beberapa wilayah hujan, ujar prakirawan cuaca Bandar Udara Supadio, Pontianak, Deby.
	Kepala Badan Penanggulangan Bencana Daerah (BPBD) Kota Pontianak Aswin Thaufiek mengatakan, salah satu kebakaran lahan terjadi di Kelurahan Kota Baru, Pontianak, Selasa. Kami baru bisa memadamkan api setelah empat jam, katanya.
	Menurut Aswin, potensi kebakaran lahan masih ada meskipun panas sudah berkurang. BPBD Kota Pontianak telah menyiapkan langkah untuk mengantisipasi kebakaran yang diperkirakan masih akan berlangsung hingga puncak musim kemarau pada Juni.
	Langkah antisipasi itu berupa simulasi pemadaman yang melibatkan pemadam kebakaran swasta dan relawan sebanyak 70 orang serta tim Taruna Siaga Bencana 20 orang.
	Titik api juga masih ditemukan di Kalteng meskipun beberapa hari terakhir hujan. Berdasarkan pantauan Kompas, di Kabupaten Gunung Mas terdapat lahan yang terbakar dan menimbulkan asap di Kota Kuala Kurun.
	Namun, menurut Kepala BPBD Kabupaten Gunung Mas M Rusdi, pihaknya belum menemukan titik api. Mungkin lahan yang terbakar itu adalah milik warga. Mereka sengaja membakar untuk membersihkannya dari semak belukar, kata Rusdi.
	Rusdi mengatakan, pihak BPBD terus mengimbau warga tidak membakar lahan agar tidak menyebabkan kabut asap dan kebakaran yang meluas. Selama ini ada sekitar 70 hektar lahan yang terbakar di Kabupaten Gunung Mas, katanya.
	Prakirawan Stasiun Meteorologi Palangkaraya, Alpon Sepriando, mengatakan, pada Selasa terpantau ada 17 titik api. Titik api tersebar di Kabupaten Kapuas, Katingan, Kotawaringin Timur, Seruyan, dan Kota Palangkaraya, kata Alfon.
	Sementara itu, di Kalimantan Timur dan Kalimantan Utara selama 2004 ini terpantau 22 titik api. Titik api ini disebabkan pembakaran lahan untuk perkebunan oleh masyarakat.
	Sejauh pantauan kami, sudah ada kesadaran dari masyarakat, yakni ketika membakar lahan, kegiatan itu mereka awasi, kata HM Faisal, Kepala Brigade Pengendalian Kebakaran Hutan Daerah Operasi Kabupaten Paser Balai Konservasi Sumber Daya Alam Kaltim. (DKA/ESA/PRA)
