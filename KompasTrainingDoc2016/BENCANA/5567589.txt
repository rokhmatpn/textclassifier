
	Kecelakaan
	Odong-odong Tertabrak Kereta Api, Satu Orang Tewas


	Kendal, Kompas Kecelakaan di pintu pelintasan kereta api kembali terjadi. Kali ini, sebuah odong-odong atau kereta mini tertabrak Kereta Api Kaligung jurusan Tegal-Semarang di pelintasan rel ganda di Desa Karanganom, Kecamatan Weleri, Kabupaten Kendal, Jawa Tengah, Rabu (19/3) sekitar pukul 14.30. Kejadian ini mengakibatkan satu orang tewas dan lima anak terluka, dua di antaranya dalam kondisi kritis.
	Saat itu, odong-odong dengan sekitar 30 penumpang, sebagian besar anak-anak, yang dikemudikan Nurul Fahmi (30) meluncur dari arah utara. Ketika tiba di pelintasan kereta api, kata Darsi (49), salah satu penumpang, pintu pelintasan ditutup dan setelah itu melintas kereta api barang dari arah timur. Setelah itu, pintu pelintasan terbuka dan odong-odong, termasuk kendaraan lainnya, melintas.
	Pas sampai di tengah, kami kaget karena ada kereta api dari arah barat. Lalu, kami langsung keluar dari odong-odong untuk menyelamatkan diri, kata Darsi.
	Darsi mengatakan, saat itu jarak kereta api dengan odong-odong tidak sampai 10 meter. Ia dan ketiga cucunya selamat. Namun, satu cucunya, Nurul (10), kaki kanannya patah. Nurul dan korban lainnya dirawat di Rumah Sakit Islam Kendal.
	Banyak yang luka, patah kaki atau tangan, saat melompat dari odong-odong. Bahkan, saya lihat ada yang melempar cucunya supaya tidak tertabrak kereta, ucap Darsi.
	Ramisih (60) yang tidak sempat keluar dari odong-odong tewas dalam kecelakaan tersebut. Bu Ramisih sudah tua. Dia duduk di depan saya, kata Darsi.
	Korban lain, Sarmi (56), warga Desa Bumiayu, Weleri, mengalami patah pada tangan kiri. Cucunya, Arikah (5), dalam kondisi kritis. Saya bingung, tangan saya harus dioperasi, cucu saya masih kritis, sedangkan orangtuanya kerja di luar negeri jadi tenaga kerja Indonesia. Saya tidak tahu bagaimana harus membayar biayanya, ujar dia.
	Kepala Kepolisian Sektor Weleri Ajun Komisaris Haryo Deko Dewo mengatakan, berdasarkan keterangan saksi, pintu pelintasan memang terbuka sebelum kereta api yang kedua lewat dari arah berlawanan. Odong-odong terseret sejauh 200 meter dari lokasi kejadian sebelum terlempar ke pinggir rel.
	Haryo mengatakan, pihaknya pernah mendapat laporan dari masyarakat terkait pengoperasian rel ganda. Pernah kejadian, kereta api lewat, tetapi palang belum ditutup. Sepertinya sistem persinyalan masih belum siap, kata Haryo.
	Namun, menurut Kepala Humas PT KAI Daerah Operasional IV Eko Budiyanto, petugas sudah menurunkan palang pintu pelintasan dan tidak membuka palang sampai kereta api berikutnya lewat dari arah berlawanan.
	Laporan yang saya terima, warga tidak sabar menunggu sehingga palang pintu diangkat dengan paksa. Padahal, jeda antara kereta pertama dan kedua hanya satu menit, kata dia.
	Di Kalimantan Barat, Anthony Chandra (17), siswa SMA Immanuel Pontianak, tewas tertabrak truk di Jalan A Yani, Pontianak, Selasa. (uti/esa)
