
	Kebakaran
	Di Tengah Banjir, Lima Rumah Hangus


	Jakarta, Kompas Lima rumah warga yang ditinggal mengungsi di Pesing Koneng, RT 012 RW 003, Kelurahan Kedoya Utara, Kecamatan Kebon Jeruk, Jakarta Barat, hangus dilalap si jago merah. Peristiwa terjadi pada Selasa (4/2) pukul 21.00. Ketua RT setempat Muhammad (50) mengatakan, api berawal dari rumahnya.
	Karena lampu di permukiman dipadamkan, pukul 18.30 saya menyalakan lilin. Saya berada sendiri di rumah, sedangkan anak-anak mengungsi di bantaran rel. Istri saya saat itu masih berjualan di Pasar Inpres, Kedoya, ujarnya saat ditemui wartawan, kemarin.
	Warga kembali mengungsi ke bantaran rel setelah hujan deras mengguyur Jakarta dari Senin malam hingga Selasa siang lalu. Pukul 20.00, lanjut Muhammad, dia ke luar rumah hendak membagi makanan bantuan kepada warga yang mengungsi. Namun, ia lupa mematikan lilin yang diletakkan di piring kecil di atas mesin VCD.
	Pukul 21.00, beberapa warga berteriak ada api di permukiman. Muhammad yang kala itu ada di pengungsian mendapat laporan, rumahnya terbakar. Ia lalu ke lokasi bersama sejumlah
	warga ke permukiman yang masih tergenang air setinggi satu meter.
	Waktu saya sampai, api sudah besar, bahkan sudah merambat ke beberapa rumah
	di sampingnya, ujar Muhammad. Ia melihat, puluhan warga sudah bekerja sama memadamkan api.
	Sementara itu, saat pulang dari pasar, Sri (48), istri Muhammad, terkejut mendapati api sudah menelan rumahnya. Bahkan, api sudah menjilat liar ke rumah di kiri dan kanan rumahnya.
	Di tengah kobaran api tersebut, Sri berniat nekat masuk ke rumahnya menerobos api untuk mengambil tabung gas. Saya khawatir tabung gas meledak dan membuat api mengamuk dan menghancurkan rumah orang, ujar Sri.
	Niat Sri nekat masuk rumah yang masih tergenang dapat dicegah oleh tetangga. Sri menangis meraung-raung sambil berulang kali meminta maaf kepada warga. Barang-barang saya habis, termasuk surat tanah, BPKB motor, dan akta kelahiran, ujar Sri.
	Matanya berkaca-kaca mengenang musibah yang baru ia lewati. Setengah jam setelah kebakaran, para petugas dan kendaraan pemadam kebakaran baru tiba.
	Itu juga sudah untung. Kalau saja para petugas dan kendaraan pemadam kebakaran tiba seperempat jam lebih lambat, api pasti sudah menghanguskan puluhan rumah lain, katanya.
	Jamilah (25), warga lain yang rumahnya berdiri di sebelah rumah Muhammad, mengatakan, saat api mulai membakar rumah Muhammad, dirinya; ibunya, Aminah (75); dan anaknya, Darlia (11), sedang berada di lantai dua rumah mereka.
	Saat itu, saya lagi berbaring dan melihat ada asap menyelusup ke rumah saya. Tidak berapa lama, saya mendengar orang-orang berteriak, kebakaran, kebakaran, dan minta
	kami keluar rumah, kata Jamilah.
	Ia turun buru-buru dan melihat ke luar, api sudah menjilat sebagian rumahnya. Saya lalu memapah paksa ibu yang sedang sakit. Saya sudah tidak berpikir menyelamatkan barang-barang. Yang penting ibu dan anak saya selamat, ujar Jamilah.
	Harta benda Jamilah, termasuk beberapa barang elektronik milik saudaranya yang dititipkan di rumahnya, hancur. Saudara saya menitipkan sebagian barangnya karena rumahnya kebanjiran, ucap Jamilah.
	Sampai Rabu sore kemarin, ratusan warga RT 012 RW 003 masih mengungsi di tenda-tenda pengungsian di bantaran rel. Genangan di rumah mereka pun belum surut.
	Berdasarkan pengamatan Kompas, tinggi genangan di beberapa lokasi di sana sebagian mencapai satu meter. Di lingkungan RT 013, tinggi air juga masih satu meter. Menurut Ketua RT Sukron, jumlah warganya yang mengungsi masih 500-an orang. Menurut dia, pada Selasa lalu permukiman di lingkungannya tergenang sampai dua meter. 
	"Barang-barang saya habis, termasuk surat tanah,BPKB Motor, dan akta kelahiran"
