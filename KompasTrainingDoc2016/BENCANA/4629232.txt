
	Asap Ganggu Penerbangan
	Siswa TK dan SD di Kota Pontianak Diliburkan


	PONTIANAK, KOMPAS - Kabut asap yang menyelimuti wilayah Kalimantan Barat akibat kebakaran lahan membuat penerbangan dari dan menuju Pontianak terlambat. Bahkan, pada Rabu (6/2) pukul 07.00, sebuah pesawat harus kembali ke Jakarta karena jarak pandang di Bandar Udara Supadio hanya 50 meter.
	Standar jarak pandang minimal 1 kilometer baru bisa memenuhi standar keselamatan penerbangan, kata Operational Services Manager Bandara Supadio, Pontianak, Usmulyani Alkadrie, Rabu, di Pontianak.
	Akibat kabut asap, kata Usmulyani, kedatangan dan keberangkatan pesawat terlambat. Aktivitas penerbangan yang normalnya dimulai pukul 06.00, Rabu, dimulai pukul 08.00, dan kemarin menjadi pukul 08.45.
	Penumpang salah satu maskapai penerbangan nasional yang baru saja tiba di Pontianak, Wili (30), menuturkan, jadwal keberangkatan dari Bandara Soekarno-Hatta, Jakarta, menuju Bandara Supadio seharusnya pukul 07.00. Namun, pesawat baru berangkat dari Bandara Soekarno-Hatta sekitar pukul 08.00. Penumpang diberi tahu penerbangan terlambat karena adanya kabut, ujar Wili.
	Kabut asap di Pontianak terjadi dalam seminggu terakhir dan semakin parah. Kemarin pagi, kabut asap pekat menyelimuti Kota Pontianak. Jarak pandang di jalan hanya sekitar 200 meter. Murid taman kanak-kanak dan sekolah dasar diliburkan agar tidak semakin terpapar asap. Sejumlah warga terlihat mengenakan masker ketika beraktivitas di luar ruangan.
	Titik api
	Prakirawan Badan Meteorologi, Klimatologi, dan Geofisika (BMKG) Bandara Supadio, Dasmin Sulviani, mengatakan, ada 68 titik api di Kalbar yang tersebar di lima kabupaten. Titik api paling banyak di Kabupaten Pontianak sebanyak 26 titik disusul Sambas 18 titik dan Kubu Raya 13 titik.
	Kabut asap juga terjadi di wilayah Kalimantan Tengah dan Riau. Namun, kabut asap tersebut tidak sampai mengganggu penerbangan.
	Prakirawan Stasiun Meteorologi Palangkaraya, Rena Trisantikawaty, mengatakan, kabut asap di Kalimantan Tengah terjadi karena terbakarnya lahan gambut dan pembukaan lahan/ladang.
	Titik api terdata muncul 27 Januari. Rena mengatakan, pada 4 Februari terdata ada 157 titik api yang tersebar di 10 kabupaten/kota. Titik api paling banyak terdapat di Kabupaten Katingan (49 titik) disusul Kotawaringin Timur 27 titik, Palangkaraya 23 titik, Gunung Mas 22 titik, serta Kotawaringin Barat dan Kapuas masing-masing 12 titik.
	Meskipun masih ada kabut asap tipis, titik api Riau berdasarkan pengamatan satelit Aqua Terra pada Kamis pukul 17.00 tercatat nol atau tidak ada. Padahal, pada Rabu, tercatat ada 261 titik api di Riau.
	Kami tidak mengetahui persis penyebab penurunan sangat tajam ini. Kemungkinan karena hari Kamis ini cuaca di atas Riau cukup ditutupi awan mendung sehingga tidak terbaca satelit, ujar Slamet Riyadi, analis cuaca BMKG Pekanbaru. (ESA/DKA/SAH)
