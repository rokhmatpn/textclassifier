
	7 Senjata Api dan Amunisi Terbakar
	15 Kamar Asrama Polres Poso Hangus


	PALU, KOMPAS - Asrama milik Kepolisian Resor Poso, Sulawesi Tengah, Minggu (23/2) dini hari, ter-bakar. Tak ada korban jiwa dalam kejadian itu, tetapi sejumlah senjata api dan amunisi hangus. Kebakar-an itu diduga karena hubungan pendek arus listrik.
	Sebanyak 15 kamar, 7 pucuk senjata api laras panjang, sejumlah amunisi, 4 unit sepeda motor milik anggota, dan perlengkapan harian anggota ludes terbakar, kata Kepala Polres Poso Ajun Komisaris Besar Susnadi saat dihubungi dari Palu, Minggu.
	Ia menambahkan, berdasarkan informasi yang dihimpun dari penghuni asrama, kebakaran diduga disebabkan hubungan pendek arus listrik. Usia bangunan asrama itu memang sudah tua. Instalasi listrik pun sudah sangat lama sehingga rentan terjadi hubungan pendek arus listrik.
	Meskipun demikian, ujar Susnadi, kepastian penyebab kebakaran itu masih didalami. Markas Besar (Mabes) Polri akan melakukan penyelidikan mendalam di lokasi kejadian.
	Saya meminta tim dari Pusat Laboratorium Forensik Mabes Polri untuk mengolah tempat kejadian perkara. Secepatnya mereka datang. Mereka khusus untuk mengidentifikasi senjata
	serta amunisi yang ikut hangus, ujar Susnadi. Namun, ia tak menyebut kapan persisnya tim Puslabfor itu datang.
	Kebakaran asrama polisi itu terjadi pada Minggu pukul 01.30 hingga 04.30 Wita. Asrama dihuni 17 anggota Polres Poso.
	30 rumah hangus
	Dari Sumatera Utara dilaporkan, sebanyak 30 rumah di permukiman padat penduduk di kawasan Listrik Atas, Kelurahan Gundaling I, Berastagi, Kabupaten Karo, Minggu dini hari, hangus terbakar. Tidak ada korban jiwa dalam kebakaran itu. Tiga rumah sengaja dihancurkan warga untuk menghindari penyebaran api sehingga total kerusakan mencapai 33 rumah.
	M Manihuruk (64), warga Listrik Atas, mengatakan, ia mendengar teriakan kebakaran sekitar pukul 02.30. Saya melihat dua rumah terbakar. Angin membuat api cepat menyebar, ucapnya. Api baru bisa dipadamkan pukul 06.00 atas gotong royong warga dan bantuan tiga mobil pemadam kebakaran.
	Akibat kebakaran itu, 156 warga kehilangan tempat tinggal dan harta benda. Menurut Kepala Lingkungan 2 Listrik Atas Ardin Simanjorang, api diduga berasal dari rumah S Sagala di pojokan kawasan permukiman. Api menyebar dibawa angin, hingga rumah satu kawasan habis. (VDL/WSI)
