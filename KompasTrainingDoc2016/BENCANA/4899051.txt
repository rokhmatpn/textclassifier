
	Ancaman Kebakaran Lahan
	Titik Api Tak Terpantau Lagi di Kalbar


	PONTIANAK, KOMPAS - Hujan lebat yang mengguyur sebagian besar wilayah Kalimantan Barat selama dua hari terakhir berdampak pada hilangnya titik api, termasuk di Kota Pontianak dan Kabupaten Kubu Raya. Kabut asap yang beberapa hari lalu masih menyelimuti Pontianak dan Kubu Raya sudah tidak tampak lagi pada Minggu (16/2).
	Pantauan kami sejak pukul 05.00 tadi titik api hilang. Tiga hari lalu titik api masih terpantau mencapai 51 titik, ujar prakirawan cuaca Bandara Supadio, Pontianak, Dina Ike, Minggu, di Pontianak. Dina menuturkan, tiga hari ke depan cuaca di Kalbar berpeluang besar hujan. Hal itu didasarkan pada suhu di permukaan air laut di perairan wilayah Kalbar yang mulai hangat, yakni berkisar 28-29 derajat celsius sejak sekitar dua hari lalu.
	Berdasarkan data Badan Meteorologi, Klimatologi, dan Geofisika Kalbar, wilayah yang berpotensi hujan antara lain Kabupaten Kubu Raya yang menjadi salah satu sumber kabut asap di Kalbar, Mempawah, Sanggau, dan Singkawang. Berdasarkan pantauan Kompas, mulai dari Kabupaten Singkawang, Mempawah, hingga Kota Pontianak pada Sabtu dan Minggu sore diguyur hujan sekitar dua jam.
	Misalnya, di wilayah Purnama, Kecamatan Pontianak Selatan, yang beberapa hari lalu selalu diselimuti kabut asap, pada Minggu sudah cerah. Demikian juga di Kubu Raya, terutama di sekitar daerah pertanian, udaranya tidak membuat sesak pernapasan lagi. Penerbangan di Bandara Supadio pun berjalan normal.
	Kepala Unit Penanggulangan Kebakaran Hutan dan Lahan Dinas Kehutanan Kalbar Boy Manuputty menambahkan, tim pemadam kebakaran yang biasanya siaga di lokasi, Minggu, sementara diliburkan karena titik api tidak terpantau lagi. Jumat lalu sebenarnya titik apinya sudah berkurang. Tinggal titik api kecil dan tidak berdampak signifikan, ujarnya.
	Namun, Boy meminta masyarakat tetap waspada. Potensi kebakaran lahan masih ada karena lahan di Kubu Raya banyak yang berupa gambut yang mudah terbakar meskipun tersulut api kecil. Jangan membakar lahan sembarangan. Sekecil apa pun api di lahan gambut, tetap berbahaya, bisa melebar apinya, katanya. Boy melanjutkan, berbagai instansi akan membahas berbagai bencana, termasuk kabut asap, di Kementerian Sosial.
