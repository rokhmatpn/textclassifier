
	Tim Kewalahan Padamkan Api
	Jambi Masih Kekurangan Selang


	PEKANBARU, KOMPAS - Proses pemadaman kebakaran lahan dan hutan di Riau mengalami kendala serius, baik lewat darat maupun udara. Dari darat, sumber air sulit ditemukan di lokasi, dan jika melalui udara, sebaran titik api terlalu luas sehingga membuat prioritas pemadaman hampir mustahil.
	Hanya hujan yang mampu memadamkan api saat ini. Hampir tidak mungkin menentukan prioritas pemadaman karena sebaran titik api sudah terlalu besar, kata Sonny Sumarsono, pilot helikopter yang disewa untuk memadamkan api di kawasan konservasi alam Cagar Biosfer Giam Siak Kecil-Bukit Batu, di Posko Penanggulangan Bencana Asap Riau, Minggu (2/3).
	Sonny juga pesimistis, penggunaan helikopter jenis Sikorsky yang dapat membawa muatan air 4 ton mampu memadamkan api pada lokasi kebakaran. Sebaliknya, Komandan Korem 031/Wirabima Brigadir Jenderal Prihadi Agus Irianto bersikukuh tetap melakukan pemadaman seperti yang sudah direncanakan. Pemadaman lewat jalur darat dan udara akan terus dilaksanakan. Kami tidak terpengaruh dengan isu (pernyataan Sonny) itu. Itu mungkin hanya argumen teknis. Tetapi, kami akan tetap jalan. Masih banyak alternatif. Satgas darat jalan terus, ujar Prihadi.
	Secara terpisah, Kepala Polsek Bukit Batu, Bengkalis, Komisaris Sazli Rais mengatakan, pihaknya baru turun dari lokasi Cagar Biosfer Giam Siak Kecil-Bukit Batu, Sabtu. Pemadaman lewat darat memang sangat sulit. Untuk mencapai lokasi itu, dari Mapolres Bukit Batu menempuh perjalanan lima jam. Kendaraan gardan ganda masih dapat dipakai, tetapi harus berjalan kaki sejauh 2 kilometer lagi untuk sampai ke lokasi perambahan.
	Terkait penerbangan, menurut Baiquni, Duty Manager Bandara Sultan Syarif Kasim II, Pekanbaru, sepanjang Minggu, 12 penerbangan (6 kedatangan dan 6 keberangkatan) mengalami penundaan karena jarak pandang hanya 800 meter.
	Di Jambi, tim pemadam juga kesulitan mengendalikan kebakaran kebun sawit di Kecamatan Kumpeh Ilir, Muaro Jambi. Jauhnya jarak antara sumber air dan lokasi-lokasi yang terbakar menyebabkan api belum berhasil dikendalikan selama lima hari terakhir. Mesin sudah cukup banyak, tetapi kami masih kekurangan selang, ujar M Zakir, Kepala BPBD Muaro Jambi.
	Sebanyak 17 titik panas di Provinsi Jambi terpantau Satelit NOAA 18. Api tersebut tersebar di Kabupaten Tanjung Jabung Timur 7 titik, Batanghari 4 titik, Tanjung Jabung Barat 3 titik, Tebo 2 titik, dan Muaro Jambi 1 titik. (SAH/ITA)
