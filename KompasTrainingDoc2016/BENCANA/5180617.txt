
	Kebakaran
	Tujuh Warga Tewas, Delapan Kios Hangus


	PEKANBARU, KOMPAS - Kebakaran di Provinsi Riau tidak hanya melanda lahan dan hutan. Jumat (28/2) dini hari, kebakaran hebat menghanguskan delapan kios, yang juga menjadi tempat tinggal, di persimpangan Jalan Soekarno Hatta dan Jalan Durian, Pekanbaru. Tujuh warga tewas dan tiga orang lainnya mengalami luka bakar akibat kejadian itu.
	Korban tewas adalah Roni (32), pemilik usaha perbaikan jok mobil; istrinya, Ningsih (26); anaknya, Neila (3); adiknya, Rahmat (20); dan pembantunya, Ipah (17). Api juga menewaskan pemilik kios ban, Makmur Munte (24), dan rekan kerjanya, Adi Manalu (20).
	Jenazah Roni, Ningsih, dan Neila ditemukan dalam keadaan berpelukan di dalam kamar dekat tempat tidur. Jenazah Rahmat dan Ipah ditemukan dalam kamar mandi. Semua korban tewas dalam keadaan hangus.
	Kami masih menyelidiki penyebab kebakaran. Semua korban dibawa ke Rumah Sakit Umum Daerah (RSUD) Arifin Achmad, Pekanbaru, untuk diotopsi, kata Kepala Satuan Reserse Kriminal Polresta Pekanbaru Komisaris Arief Fajar S, Jumat.
	Berdasarkan keterangan yang dihimpun dari lokasi, api diduga bermula dari warung milik Roni. Api dengan cepat membakar bangunan yang sebagian besar terbuat dari kayu itu. Api menyambar kios Makmur dan membakar ban bekas yang ada di dalamnya.
	Heri, seorang saksi mata, menceritakan, semula dia melihat api di kios Roni. Dia dan beberapa orang yang kebetulan melintas coba membangunkan pemilik
	kios. Beberapa penghuni cepat terbangun dan ikut berteriak agar tetangga mereka bangun untuk menghindar dari api.
	Api sangat cepat berkobar karena kondisi cuaca begitu kering. Kami tidak mampu memadamkan api, kata Heri.
	Warga yang berkumpul sebenarnya sudah berupaya keras memadamkan api dengan menggunakan ember dan alat seadanya. Namun, kobaran api terlalu besar, sementara tidak seorang pun yang mengetahui nomor telepon dinas pemadam kebakaran.
	Beberapa orang berangkat ke kantor dinas pemadam kebakaran dengan mengendarai sepeda motor. Mobil pemadam kebakaran baru tiba sekitar 30 menit setelah kejadian. Api dapat dikendalikan setelah pemadam kebakaran berjuang selama lebih dari satu jam.
	Wulan Siregar (26), pemilik kios jahit pakaian yang terbakar, mengungkapkan, ia dan keluarganya selamat karena cepat terbangun saat mendengar teriakan dari luar. Saat keluar, api belum terlalu besar.
	Saya tidak menyangka api begitu cepat membesar. Tidak lama kemudian, saya melihat api ada di mana-mana. Kami tidak dapat menyelamatkan harta benda yang kami kumpulkan selama ini. Yang tersisa hanya pakaian di badan, kata Wulan.
	Di Kamar Mayat RSUD Arifin Achmad, F Munte, anggota keluarga Makmur Munte, mengungkapkan, Makmur dan Adi bersahabat karena keduanya berasal dari daerah yang sama, Sidikalang, Sumatera Utara, dan merantau ke Pekanbaru. (SAH)
