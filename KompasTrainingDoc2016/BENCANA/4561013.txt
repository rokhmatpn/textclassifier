	Titik Api Kembali Bermunculan di Jambi


	Jambi, Kompas - Titik api kembali bermunculan di wilayah Jambi setelah hujan tidak turun selama sepekan terakhir. Sebaran titik api yang terpantau dari lokasi lahan terbakar tersebut dipicu juga oleh pergerakan angin yang cukup tinggi.
	Data Satelit NOAA 18 yang diolah Dinas Kehutanan Provinsi Jambi menyebutkan, sebanyak 9 titik api tersebar pada 2 kabupaten. Ada 7 titik di Kabupaten Tebo dan 2 titik di Tanjung Jabung Barat, kata petugas pengolah data titik api (hotspot) Dishut Provinsi Jambi, Donny Osmond, Senin (3/2) di Kota Jambi.
	Sebagian besar kebakaran lahan ini berada dalam areal kebun masyarakat. Sisanya di kawasan hutan tanaman industri yang jadi area perambahan liar. Namun, kemunculan titik-titik api baru belum dianggap mengancam.
	Kepala Bidang Perlindungan Hutan Dishut Provinsi Jambi Agus Sriyanta mengatakan, kemunculan titik-titik api baru terjadi satu atau dua hari terakhir. Pihaknya belum akan turun ke lokasi kebakaran untuk memadamkan, tetapi tetap terus memantaunya. Upaya pemadaman baru dilakukan apabila api di lokasi bertahan minimal 2 hari.
	Pihak Dishut Jambi telah berkoordinasi dengan Badan Meteorologi, Klimatologi, dan Geofisika Jambi. Diperoleh informasi bahwa curah hujan masih cukup tinggi hingga April. Kondisinya belum perlu dikhawatirkan karena hujan diperkirakan masih akan sering turun dalam beberapa waktu ke depan, ujarnya.
	Terkait munculnya titik api baru, Koordinator Bidang Pengkajian dan Informasi BMKG Jambi, Kurnianingsih mengaku, kondisi itu dipicu pergerakan angin cukup tinggi. Kecepatan angin hingga 18 kilometer per jam, ujarnya.
	Wilayah Jambi secara umum masih musim hujan, tetapi tidak menutup kemungkinan hujan tak turun dalam rentang waktu tertentu. Suhu bervariasi dari 23 hingga 32 derajat celsius.
	Jika tak hujan dalam beberapa hari, masyarakat diminta mewaspadai angin kencang disertai hujan deras pada hari berikutnya. Suhu panas memicu pembentukan awan kumulonimbus sehingga potensi berikutnya adalah angin kencang disertai petir dan diikuti hujan deras, kata Kurnianingsih. (ITA)
