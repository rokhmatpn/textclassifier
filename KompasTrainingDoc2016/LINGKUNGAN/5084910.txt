
	Kabut Asap
	Penerbangan di Pontianak Tertunda


	PONTIANAK, KOMPAS - Kabut asap pekat akibat pembakaran lahan kembali menyelimuti Kota Pontianak dan Kubu Raya, Kalimantan Barat, Minggu (23/2). Akibatnya, penerbangan dari dan menuju Bandar Udara Supadio, Pontianak, tertunda sekitar 1 jam 30 menit karena jarak pandang hanya sekitar 800 meter.
	Berdasarkan pantauan Kompas, kabut asap pekat menyelimuti Kota Pontianak dan sekitar Kubu Raya sejak Sabtu sekitar pukul 21.00. Jarak pandang malam itu hanya sekitar 800 meter. Asap bahkan masuk hingga ke dalam rumah dan menyesakkan pernapasan. Kabut asap semakin pekat pada Minggu pagi. Penerbangan dari dan menuju Bandara Supadio yang seharusnya dimulai pukul 06.00 ditunda hingga pukul 07.30. Di jalan raya, untuk menjaga keamanan, sebagian besar pengendara mobil dan sepeda motor menghidupkan lampu utama kendaraan.
	Bercampur embun
	Kabut asap ini semakin pekat bercampur embun karena adanya penguapan permukaan tanah. Meskipun terjadi keterlambatan penerbangan, tetapi tak sampai membuat penumpang membeludak, ujar Operation Services Manager Bandara Supadio Usmulyani Alkadrie, Minggu.
	Prakirawan Bandara Supadio, Dasnian Sulviani, mengatakan, penerbangan kembali beroperasi setelah jarak pandang di bandara 1.500 meter. Titik api pada Minggu, lanjut Dasnian, tinggal sembilan titik dibandingkan dengan Sabtu yang mencapai 10 titik.
	Titik api pada Minggu didominasi dari daerah Kabupaten Pontianak, yakni enam titik api, kemudian Bengkayang tiga titik api, kata dia.
	Berdasarkan pantauan Kompas, asap tampak mengepul dari lahan di daerah-daerah yang rawan kebakaran di Kota Pontianak, misalnya daerah Purnama dan Paris 2. Asap disebabkan api pembakaran sampah dan pembakaran lahan untuk bercocok tanam di lahan gambut.
	Kepala Badan Penanggulangan Bencana Daerah (BPBD) Kota Pontianak Aswin Thaufiek, Minggu, mengatakan, tidak ada kebakaran besar di Kota Pontianak. Kabut asap yang pekat diperkirakan disebabkan titik-titik api kecil di tepi kota yang tidak terpantau citra satelit.
	Aswin memperkirakan, asap di Kota Pontianak terbawa angin dari lahan terbakar di daerah Sungai Pinyuh dan Sungai Kunyit, Kabupaten Pontianak. Sungai Pinyuh ada tiga titik api dan Sungai Kunyit ada satu titik api, kata dia. Sementara itu, meski hujan dengan intensitas ringan dan sedang telah mengguyur sebagian besar wilayah Kalimantan Tengah, tetap saja masih ada titik api. Pada Sabtu, terpantau ada dua titik api di Sampit, Kotawaringin Timur.
	Titik api terpantau di Kecamatan Tualan Hulu dan Bagendang. Luasan lahan (terbakar) belum diketahui. Situasi kabut asap masih normal, kata Kepala BPBD Kabupaten Kotawaringin Timur Lumban Gaol saat dihubungi dari Palangkaraya. (ESA/DKA)
