
	Superioritas Samator Rontok
	Pertamina Putri Perpanjang Kemenangan Sempurna


	Solo, Kompas - Superioritas tim putra Surabaya Samator rontok dalam penampilan keempat mereka. Mereka menyerah dari juara bertahan Palembang Bank SumselBabel, 21-25, 19-25, 25-11, 14-25, di seri kedua kompetisi bola voli BSI Proliga di Gelanggang Olahraga Sritex Arena, Solo, Minggu (19/1).
	Dengan segudang pemain nasional dan setter asing berkualitas, Samator difavoritkan juara. Namun, kemarin, blok mereka rapuh, beberapa kali seperti tertidur ketika menerima servis dan monoton dalam smes.
	Pelatih Samator Ibarsjah DT menyebut pemainnya seperti tidak bermain voli. Itu yang kemarin saya takutkan. Banyak pemain muda di tim kami, tetapi seperti bukan bermain voli. Tosser sudah bagus, spike-nya terlalu menggebu-gebu sehingga mudah hilang (poin), paparnya.
	Pada set pertama, permainan masih imbang. Bank SumselBabel memetik banyak poin dari smes pemain asingnya, Ismael Luke Fernandez. Servis lompat pemain Kuba itu juga sangat keras dan mematikan.
	Libero Samator Veleg Dhani Ristan Krisnawan beberapa kali terjengkang menerima servis Fernandez. Meski begitu, di set pembuka itu, Samator masih terlihat berpeluang memperpanjang tiga kemenangan sebelumnya.
	Modal mereka adalah lima pemain nasional dan setter  Kuba Raydel Hierrezuelo Aguirre. Umpan Aguirre matang, bola tipuannya maut dan mengejutkan. Dalam situasi itu, pengumpulan poin kedua tim terus berimpitan. Setelah ketat hingga 20-22, Bank SumselBabel menang 25-21.
	Di set kedua, kelemahan blok Samator kian mencolok. Jangankan mengantisipasi bola hantaman Fernandez yang keras kencang bak taifun, menjinakkan serangan pemain lokal M Khasoni Mufid pun mereka tak berdaya.
	Itu yang membuat Aguirre frustrasi. Ia beberapa kali mengingatkan rekan-rekannya, merasa bola-bola serangan lawan dibiarkan menghunjam dirinya. Ia merasa tidak ada cover dan sangat marah, ujar Gonzalez Toiran Riedel Alfonso, pemain Samator yang juga dari Kuba.
	Bank SumselBabel selalu unggul. Mulai 10-5, 14-8, 20-12, hingga mengunci set kedua dengan 25-19. Setter Samator sering jadi momok lawan. Namun, setelah gagal passing, ia mudah emosi. Jika ia emosi, permainan mereka kacau, kata Agung Seganti, pemain nasional di Bank SumselBabel.
	Harapan Samator untuk bangkit muncul di set ketiga. Mereka seperti menemukan resep mengeblok smes lawan. Duet Randu-Rendy berhak dipuji untuk itu. Penonton memberi aplaus meriah kepada keduanya. Di set itu, Samator menang 25-11.
	Namun, Samator kembali kehilangan irama di set keempat. Para pemain seperti mengantuk, membiarkan bola servis lawan masuk di lapangan mereka. Setelah tertinggal 7-13, 11-20, Samator menyerah 14-25. Berakhir sudah superioritas mereka.
	Pertamina lari kencang
	Di putri, Jakarta Pertamina Energi memperpanjang rekor tanpa kehilangan satu set pun dalam kemenangan keempat saat memukul Jakarta Electric PLN, 25-12, 25-23, 25-10. Diperkuat enam pemain nasional, tim asuhan pelatih Octavian itu bak berlari secepat mungkin untuk merebut tiket babak empat besar.
	Kemenangan 3-0 juga dipetik Jakarta Popsivo PGN atas Gresik Petrokimia dengan 26-24, 25-11, 25-22. Adapun Jakarta Bank DKI harus bersusah payah memetik kemenangan perdana atas Jakarta BNI dengan 46 25-14, 25-15, 26-28, 19-25, 15-7.
	Bank DKI sebenarnya tinggal butuh satu poin pada set ketiga untuk menang 3-0. Namun, sampai poin 24, mereka terkejar. Lewat Sifa Mutia Beladina yang eksplosif dan inspiratif, BNI 46 menyalip pada poin 26-25 hingga akhirnya menang 28-26.
	Masih dengan inspirasi Mutia, BNI 46 kembali menang 25-19 di set keempat. Pada set penentuan, Mutia habis. Habis pula motor dan inspirasi permainan mereka. Bank DKI pun menang. (SAM)
