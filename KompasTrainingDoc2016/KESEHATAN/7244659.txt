
	KESEHATAN
	Menanti Obat Hepatitis C Terjangkau


	Oleh: DR SAMSURIDJAL DJAUZI
	Semasa SMA saya ikut-ikutan menyuntikkan narkoba. Saya tertular hepatitis C. Saya harus keluar-masuk pusat rehabilitasi. Sekarang saya kuliah di semester delapan sebuah universitas. Umur saya 23 tahun. Status hepatitis saya sekarang kronis. Menurut dokter jika dibiarkan dalam sepuluh atau lima belas tahun lagi, ada kemungkinan menjadi sirosis hati, bahkan kanker hati. Menurut informasi, terapi hepatitis C adalah suntikan interferon dan biayanya amat mahal. Terapi enam bulan dapat mencapai Rp 60 juta rupiah kadang katanya terapi perlu satu tahun. Jadi amat mahal dan tidak terjangkau oleh saya ataupun oleh keluarga saya. Apa yang dapat saya lakukan agar hepatitis kronik saya tidak bertambah parah? Apakah terapi hepatitis C kronik akan dapat dinikmati masyarakat dengan harga murah seperti halnya terapi HIV? Apakah untuk mendapat pekerjaan status hepatitis C saya akan menjadi halangan. Bagaimana jika saya menikah nanti, apakah akan menularkan kepada istri saya?
	C di M
	Hepatitis C memang merupakan penyakit yang cukup sering dijumpai di masyarakat. Sekitar 0,87 persen penduduk kita tertular hepatitis C. Pada kelompok yang pernah menggunakan narkoba, jika menggunakan jarum secara bersama risiko tertular hepatitis C dapat mencapai 70 persen. Berbeda dengan hepatitis B, risiko hepatitis C menjadi kronik jauh lebih besar. Jadi meskipun kekerapan hepatitis C lebih rendah, penderita hepatitis C kronik di Indonesia sekitar 1 juta orang.
	Penderita hepatitis C kronik perlu dipantau dengan cermat. Pemantauan meliputi pemantauan klinis, laboratorium, USG, Fibroscan, bahkan jika perlu biopsi hati. Melalui pemeriksaan ini dapat ditetapkan kelompok yang memerlukan terapi. Mereka yang memerlukan terapi sekarang ini mendapat suntikan interferon sekali seminggu dan obat ribavirin bentuk kapsul setiap hari. Pengobatan dapat enam bulan atau setahun, bergantung pada genotipe virus dan adanya HIV. Hasil terapi interferon cukup menggembirakan. Jika terapi berhasil, virus hepatitis C tidak lagi ditemukan. Sebanyak 50-80 persen kasus dapat mencapai keadaan ini. Namun, sebagian penderita ada juga yang kambuh sehingga memerlukan terapi ulangan atau terapi tambahan.
	Terapi tambahan untuk hepatitis C untuk meningkatkan keberhasilan terapi standar juga sudah tersedia di Indonesia. Terapi berupa kapsul yang diberikan per oral. Namun, terapi ini juga mahal. Jadi secara keseluruhan terapi hepatitis C memberi harapan, tetapi harga obat masih amat mahal. Kita berharap pada masa depan akan ada obat hepatitis C yang lebih sederhana pemakaiannya dan juga lebih terjangkau. Namun untuk itu, masyarakat masih harus sabar menunggu. Dulu, program Askes membiayai terapi hepatitis C, kita berharap program BPJS juga akan mampu membiayai terapi hepatitis C sehingga lebih banyak penderita yang dapat ditolong.
	Penularan
	Penularan hepatitis C melalui cairan tubuh. Hepatitis C dapat menular lewat hubungan seksual meski risiko penularan HIV jauh lebih tinggi. Ibu hamil yang positif hepatitis C berisiko menularkan kepada bayinya sekitar 7 persen. Hingga saat ini belum ada vaksin hepatitis C sehingga jika akan berkeluarga, kita berharap Anda dapat menjalani terapi hepatitis C sehingga virus hepatitis C di tubuh Anda dapat ditekan serendah mungkin (tidak terdeteksi). Penderita hepatitis C dianjurkan tidak menyumbangkan darah atau organ tubuh. Dalam skrining tenaga kerja ada perusahaan yang melalukan skrining hepatitis dan ada yang tidak.
	Perhimpunan Spesialis Penyakit Dalam Indonesia pernah menyurati Menteri Tenaga Kerja agar penderita hepatitis (hepatitis B maupun C) tidak didiskriminasi dalam pekerjaan. Ini berarti penderita hepatitis yang keadaan klinisnya baik dapat bekerja seperti temannya yang lain. Beberapa negara, misalnya Australia, telah menjalankan program terapi hepatitis C. Ini berarti biaya pengobatan yang mahal tersebut dibiayai pemerintah. Kita berharap suatu waktu kemampuan Pemerintah Indonesia juga meningkat sehingga terapi hepatitis C juga akan dibiayai pemerintah. Sementara itu, bagi penderita yang mampu, sebenarnya obat hepatitis C sudah tersedia di Indonesia, tetapi biaya pengobatan harus ditanggung sendiri.
	Kita berharap produsen obat hepatitis C dapat memberikan harga yang jauh lebih murah bagi Indonesia karena pendapatan rakyat Indonesia baru sekitar 3.000 dollar AS setahun. Jangan sampai harga obat hepatitis C di Amerika atau di Eropa disamakan dengan harga di Indonesia.
	Sudah tentu masyarakat perlu informasi tentang cara mencegah penularan hepatitis C. Hepatitis C ditularkan melalui cairan tubuh, berarti dapat menular melalui transfusi darah, jarum suntik, hubungan seksual, dan lain-lain. Hepatitis C juga berpotensi ditularkan melalui tato, tindik, atau alat cukur. Namun, penularan yang paling sering di negeri kita adalah melalui penggunaan jarum suntik bersama. Sebagian itu juga mulai banyak pengguna narkoba suntikan yang berhenti dan ikut dalam program substitusi metadon. Melalui program ini pengguna narkoba suntikan dapat berhenti dan rasa kecanduannya dapat diatasi dengan menggunakan metadon.
	Anda tentu saja dapat menikah, tetapi harus hati-hati agar istri tidak tertular. Jika belum mendapat terapi, salah satu upaya yang dapat dilakukan adalah dengan menggunakan kondom. Namun, jika virus hepatitis C tidak terdeteksi setelah terapi, Anda dapat melepas kondom sehingga dapat mempunyai anak seperti juga pasangan lain.
	Anda benar, masyarakat Indonesia menantikan obat hepatitis C yang sederhana dan mudah dijangkau. Penelitian masih berjalan terus, tetapi diperkirakan hasil penelitian obat hepatitis C yang baru masih akan mahal. Karena itu, kepedulian pemerintah, kalangan bisnis, terutama produsen obat hepatitis C amat diharapkan agar puluhan ribu, bahkan ratusan ribu penderita hepatitis C kronik dapat dicegah menjadi sirosis hati atau kanker hati.
