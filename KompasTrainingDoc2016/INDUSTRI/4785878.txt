
	Jalur KA Selesai 2014
	Kota Padang-Bandara Minangkabau Terhubung


	PADANG, KOMPAS - Jalur kereta api yang menghubungkan Kota Padang dengan Bandar Udara Internasional Minangkabau, Sumatera Barat, akan selesai dikerjakan tahun ini. Ini merupakan jalur kereta api bandara yang kedua setelah jalur kereta api Bandara Kualanamu, Sumatera Utara.
	Kepastian selesainya proyek senilai Rp 126 miliar itu disampaikan Wakil Menteri Perhubungan Bambang Susantono saat meninjau kesiapan jalur kereta api (KA) ke Bandara Internasional Minangkabau (BIM) di Padang, Rabu (12/2). Bambang didampingi Gubernur Sumatera Barat Irwan Prayitno beserta jajarannya.
	Pada kesempatan itu dilakukan uji coba railbus atau kereta api jarak pendek yang nanti digunakan menuju BIM. Uji coba dari Stasiun Simpang Aru, Kota Padang, hingga Stasiun Duku, Padang Pariaman, tersebut menjadi rangkaian Rapat Koordinasi Teknis Badan Penelitian dan Pengembangan Kementerian Perhubungan yang diselenggarakan di Padang pada 11-12 Februari.
	Jalur kereta api ke BIM tersebut nanti akan menjadi bagian dari jaringan rel Trans-Sumatera Railways yang ditargetkan tersambung 4-5 tahun ke depan.
	Saat ini, pengerjaan jalur kereta api sepanjang 4,2 kilometer dari Stasiun Duku di Padang Pariaman ke BIM tersebut sudah mulai dilakukan. Proyek ini meliputi penyiapan jalur rel hingga pemasangan rel, termasuk jembatan, serta perbaikan stasiun dan persinyalan.
	Kami ingin kereta api menjadi salah satu moda transportasi tulang punggung di Indonesia, termasuk Sumatera Barat. Khusus Sumatera Barat, selain menyelesaikan jalur menuju bandara, kami akan mengaktivasi ruas-ruas jalur yang sudah ada, kata Bambang.
	Ia menambahkan, langkah tersebut diharapkan dapat membuat masyarakat meninggalkan kendaraan pribadi dan beralih ke kereta api. Dengan begitu, beban jalan bisa dikurangi sehingga kemacetan yang mulai terjadi, termasuk di Sumatera Barat, bisa berkurang. Efisiensi pergerakan pun bisa meningkat. Selain itu, konsumsi bahan bakar minyak dan polusi bisa lebih rendah.
	Pembebasan lahan
	Bambang mengatakan, khusus untuk pengerjaan jalur kereta api menuju BIM, pihaknya akan sangat terbantu jika pemerintah daerah segera menyelesaikan pembebasan lahan yang masih tersisa.
	Direktur Lalu Lintas dan Angkutan Kereta Api Direktorat Jenderal Perkeretaapian Kementerian Perhubungan Hanggoro Budi Wiryawan berharap pemerintah daerah bisa menuntaskan pembebasan lahan hingga Juli 2014 sehingga target penyelesaian jalur pada akhir Desember 2014 bisa tercapai.
	Kepala Dinas Perhubungan, Komunikasi, dan Informatika Sumatera Barat Mudrika mengungkapkan, pihaknya tengah menyelesaikan pembebasan lahan yang dananya berasal dari APBD sebesar Rp 36 miliar. Sekitar 80 persen lahan sudah dibebaskan. Namun, upaya ini akan cepat selesai jika koordinasi antara Bupati Padang Pariaman dan Badan Pertanahan Nasional juga lebih cepat dilakukan.
	Kerja sama kedua pihak tersebut sangat diharapkan karena pembebasan lahan memang tugas mereka. Pemerintah provinsi menyiapkan dana untuk itu, ujarnya. (ZAK)
