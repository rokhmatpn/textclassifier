
	Angkutan KA Diminati
	Angkutan Barang di Jalan Raya Sebaiknya Dikurangi


	Jakarta, Kompas - Volume barang yang diangkut menggunakan kereta api pada Januari 2014 mencapai 2,3 juta ton, naik 2,85 persen dibandingkan Desember 2013. Kenaikan volume terjadi akibat jalan rusak. Namun, hal ini bisa menjadi momentum migrasi angkutan ke kereta api.
	Di Pulau Jawa, kenaikan volume angkutan barang terjadi di luar Jakarta, Bogor, Depok, Tangerang, dan Bekasi (Jabodetabek), dari 667.000 ton pada Desember 2013 menjadi 760.000 ton pada Januari 2014. Deputi Bidang Statistik Produksi Badan Pusat Statistik Adi Lumaksono menjelaskan, faktor jalan rusak akibat bencana dan cuaca menjadi penyebab migrasi ke kereta api.
	Tak hanya angkutan barang, angkutan penumpang juga mengalami kenaikan pada Januari. Pada Desember 2013, total penumpang kereta api 20,8 juta orang. Pada Januari 2014 naik menjadi 20,9 juta penumpang, kata Adi dalam konferensi pers di Jakarta, Senin (3/3).
	Di Sumatera, angkutan barang menggunakan kereta api termasuk tinggi kendati ada penurunan pada Januari 2014. Pada Desember 2013, volume barang yang diangkut dengan kereta api 1.579 ton, lalu turun menjadi 1.550 ton pada Januari 2014.
	Jumlah penumpang kereta api didominasi penumpang pelaju di Jakarta, Bogor, Depok, Tangerang, dan Bekasi. Pada Januari 2014, persentase jumlah penumpang pelaju 14,954 juta orang atau 71,65 persen dari total penumpang kereta api di Jawa.
	Data Badan Pusat Statistik menunjukkan, penumpang kereta api di luar Jabodetabek pada Januari 2014 berjumlah 5,522 juta orang, naik 8,47 persen dibandingkan jumlah penumpang pada Desember 2013 yang mencapai 5,091 juta orang.
	Kerusakan jalan juga memicu kenaikan volume angkutan barang menggunakan kapal laut. Pada Desember 2013, volume angkutan barang menggunakan kapal laut sebanyak 17,6 juta ton, naik pada Januari 2014 menjadi 18,7 juta ton.
	Migrasi angkutan barang di luar bahan kebutuhan pokok harus terus didorong untuk meningkatkan efisiensi. Sejumlah riset menyebutkan, biaya angkutan barang melalui jalan raya di Indonesia mencapai Rp 4.000 per kilometer, sementara angkutan kereta api hanya Rp 1.800 per kilometer.
	Biaya angkutan barang menggunakan sarana jalan raya di Indonesia jauh lebih besar dibandingkan Malaysia dan Thailand yang hanya sekitar Rp 1.500 per kilometer.
	Lebih efisien
	Pengamat transportasi dari Universitas Katolik Soegijapranata, Semarang, Jawa Tengah, Joko Setijowarno, menjelaskan, angkutan jalan raya sebaiknya dikhususkan untuk komoditas ringan atau bahan makanan pokok.
	Untuk komoditas di luar bahan makanan atau barang berat sebaiknya berangsur-angsur digeser ke kereta api atau kapal laut. Selain menekan biaya, juga bisa mengurangi potensi kerusakan jalan yang terus-menerus menjadi persoalan di Indonesia, kata Joko.
	Truk hanya cocok untuk angkutan barang ringan dengan jarak maksimal 400 kilometer. Dengan jarak lebih dari 400 kilometer, angkutan truk sudah tak efisien, apalagi jika barang yang diangkut berat, kata Joko.
	Waktu tempuh angkutan barang menggunakan kereta api juga jauh lebih pendek dibandingkan menggunakan truk. Salah satu contoh adalah angkutan barang menggunakan truk dari Surabaya ke Jakarta. Waktu tempuh menggunakan truk bisa tiga hingga empat hari. Sementara waktu tempuh angkutan barang menggunakan kereta api dari Jakarta ke Surabaya atau sebaliknya hanya maksimal 20 jam.
	Joko menilai, selama ini angkutan kereta api jarak jauh hanya menarik bagi penumpang pada akhir pekan. Selain yang ingin berlibur, penumpang pada akhir pekan umumnya merupakan pekerja di Jakarta yang pulang untuk menengok keluarga.
	Kereta api juga baru sebatas alternatif, bukan pilihan utama bagi penumpang. Penumpang yang semula naik pesawat terbang pindah ke kereta api jika harga tiket pesawat terlalu mahal atau tidak ada tempat duduk tersisa. Menurut Joko, hal itu terjadi karena kecepatan kereta api di Indonesia tak bisa mengejar kebutuhan waktu penumpangnya.
	Banyak penumpang butuh moda transportasi dengan waktu tempuh pendek, sementara kereta api Indonesia belum bisa menjawabnya. Infrastruktur dan sarana perkeretaapian belum memungkinkan menambah kecepatan laju kereta api sehingga waktu tempuh lebih pendek. Ini menjadi pekerjaan rumah pemerintah dan PT Kereta Api Indonesia, kata Joko. (AHA)
