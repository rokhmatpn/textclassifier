	Dua Kereta Api Sambungkan Jalur Kereta di Jateng


	Semarang, Kompas - PT Kereta Api Indonesia meluncurkan dua kereta api baru di Jawa Tengah, yaitu KA Kalijaga yang melayani rute Solo-Semarang dan KA Kamandaka yang melayani rute Purwokerto-Semarang. Moda ini diharapkan dapat menjadi alternatif transportasi publik yang cepat dan nyaman.
	Dengan adanya dua kereta api tersebut, Jateng kini dikelilingi oleh jalur KA, antara jalur utara dan selatan juga terhubung. Secara keseluruhan, dengan tambahan dua rute tersebut, sejak 2013 PT KAI sudah menambah 10 rute kereta api baru.
	Kereta Api Kalijaga diluncurkan Sabtu (15/2) di Stasiun Purwosari, Solo, oleh Direktur Komersial PT KAI Sulistyo Wimbo Hardjito, Gubernur Jateng Ganjar Pranowo, dan Wakil Menteri Perhubungan Bambang Susantono. Kereta api itu kemudian berangkat sesuai jadwal, yaitu pukul 05.28, menuju Semarang dan tiba di Stasiun Semarang Poncol pukul 07.58, dan kembali ke Solo pada pukul 09.05.
	Kereta Api yang menggunakan rangkaian KA Bengawan yang idle (tidur) tersebut melintas melalui beberapa stasiun, antara lain Solo Balapan, Gundih, Kedung Jati, Semarang Tawang, hingga Semarang Poncol. Jalur tersebut sebelumnya dilayani oleh KA Banyubiru dan KA Pandanwangi yang tiga tahun terakhir berhenti beroperasi karena minimnya penumpang.
	Wimbo mengatakan, dua KA tersebut akhirnya tidak diminati karena waktu tempuhnya lama, mencapai 4 jam. Waktu yang sama ditempuh melalui jalan raya Semarang-Solo yang lalu lintasnya sangat padat, sehingga penumpang lebih memilih naik bus atau kendaraan pribadi.
	Jika sebelumnya rute yang dibuat tidak berdasarkan kebutuhan penumpang, sekarang pola itu diubah. Kami mengikuti kebutuhan penumpang, terutama soal waktu. Kami optimistis kereta api ini pasti banyak diminati, ujar Wimbo.
	Karena itu pula, KA Kamandaka rute Purwokerto-Semarang akan diluncurkan pada Senin (17/2). KA tersebut akan berangkat dari Purwokerto pukul 05.00, kemudian tiba di Tegal pukul 07.00 dan di Semarang pukul 10.00. Kebutuhan rute Purwokerto-Tegal sangat tinggi, karena itu waktu disesuaikan pukul 07.00 kereta api tiba di Tegal.
	Salah seorang penumpang, Kumala (22), menyambut positif penghidupan kembali jalur Solo-Semarang karena ada alternatif transportasi. Dia berharap frekuensi kereta api tersebut ditambah, tidak hanya satu kali pergi pulang Solo-Semarang.
	Kepadatan berkurang
	Ganjar mengapresiasi apa yang dilakukan PT KAI. Dengan beroperasinya KA Kalijaga, diharapkan kepadatan lalu lintas di jalan Semarang-Solo dapat berkurang.
	Selain mengurangi kepadatan jalan raya, Bambang mengatakan, moda transportasi kereta api lebih menghemat energi, mengurangi polusi, dan keberadaan jalan akan lebih efisien. Ini adalah salah satu wujud kebangkitan kembali kereta api. Revitalisasi jalur-jalur yang mati akan terus dilakukan, ujarnya.
	Dalam seminar Peran Kereta Api dalam Pengembangan Kawasan Provinsi Jawa Tengah bersama Masyarakat Transportasi Indonesia (MTI) di Semarang, Bambang mengatakan, arus barang dan penumpang di Jawa, tidak bisa lagi hanya mengandalkan moda transportasi darat.
	Pengurangan beban transportasi darat di Pulau Jawa menjadi roh rencana strategis dalam lima tahun ke depan Kementerian Perhubungan. Tujuannya supaya moda transportasi bisa berimbang antara jalan darat, laut, dan kereta api, katanya. (UTI/WHO)
