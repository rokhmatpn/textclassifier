
	LOGISTIK
	Interkoneksi Antarmoda Tekan Biaya Logistik


	Bandar Lampung, Kompas Menteri Badan Usaha Milik Negara Dahlan Iskan meresmikan revitalisasi jalur kereta api menuju Pelabuhan Panjang, Bandar Lampung, dan modernisasi Terminal Curah Kering di Pelabuhan Panjang. Interkoneksi antara pelabuhan dan jalur kereta api diyakini dapat menekan biaya logistik dari Pelabuhan Panjang menuju beberapa daerah di Sumatera ataupun sebaliknya.
	General Manager Pelabuhan Panjang Doso Agung dalam sambutannya di Bandar Lampung, Jumat (24/1), mengatakan, revitalisasi jalur kereta api akan membawa banyak keuntungan.
	Dengan masuknya kereta api ke dalam wilayah pelabuhan, biaya logistik dapat ditekan, kecelakaan dan kerusakan jalan bisa dikurangi, serta dapat menjangkau konektivitas di beberapa daerah di Sumatera, katanya.
	Doso berharap proyek revitalisasi bisa diteruskan hingga menembus luar Palembang. Hingga saat ini, PT Pelabuhan Indonesia II dan PT Kereta Api Indonesia baru menyambungkan jalur kereta dari Pelabuhan Panjang menuju Tanjung Enim Baru, Sumatera Selatan, dan Tanjung Siapi- api, Sumsel, ataupun sebaliknya.
	Executive Vice President Divisi Regional 3 Sumatera Selatan Suseno menjelaskan, angkutan logistik menggunakan kereta api lebih memiliki daya angkut lebih banyak daripada menggunakan truk. Satu truk mengangkut beban 19 ton, sedangkan satu gerbong kereta mampu mengangkut 50 ton. Apabila dalam sekali perjalanan, lokomotif mampu menarik 60 gerbong, itu berarti ada 3.000 ton yang diangkut dalam sekali jalan, ujarnya.
	Selain itu, biaya yang dikeluarkan apabila menggunakan kereta api juga bisa ditekan. Suseno mengatakan, distribusi logistik menggunakan truk memakan biaya Rp 700 per ton per kilometer. Adapun menggunakan kereta, biayanya hanya sekitar Rp 500 per ton per kilometer.
	Saat ditanya mengapa hal ini tidak diterapkan di Pulau Jawa yang biasa mengandalkan jalur pantura, Menteri BUMN Dahlan Iskan mengatakan, hal itu tidak mudah. Ia mengatakan, Pelabuhan Tanjung Priok, Jakarta, juga memiliki akses jalur kereta api.
	Namun, jalur itu belum tersambung ke pelabuhan. Jaraknya masih kurang 400 meter untuk dapat tersambung dengan pelabuhan, ujarnya. (GER/ARN)
