
	Kurangi Beban Jalan
	Distribusi Logistik Harus Manfaatkan Jalur Laut dan Kereta Api


	SEMARANG, KOMPAS  Penataan distribusi logistik harus dibenahi. Momentum pembenahan jembatan timbang hanya salah satu dari upaya besar yang harus dilakukan. Ke depan, distribusi logistik tidak bisa hanya mengandalkan jalan raya, tetapi juga harus melalui jalur lain, seperti laut dan rel kereta api, untuk mengurangi beban jalan.
	Pakar transportasi dari Universitas Katolik Soegijapranata Semarang, Djoko Setijowarno, Selasa (20/5), di Kota Semarang, Jawa Tengah, mengatakan, berbagai pelanggaran di jembatan timbang menjadi cermin buruknya penataan logistik selama ini.
	Idealnya, pengiriman barang dengan truk melalui jalan raya adalah untuk jarak maksimal 500 kilometer. Selebihnya, penghitungan biaya tidak akan masuk. Saat ini, banyak barang dari Jawa Timur dikirim ke Jakarta, bahkan dari Bali ke Jakarta, lewat jalan raya. Tidak mengherankan banyak truk kelebihan muatan karena mereka tidak akan untung kalau tidak lebih muatannya, kata Djoko.
	Sementara itu, kata Djoko, moda transportasi lain kurang dilirik. Dengan kapal laut yang seharusnya bisa lebih cepat ketimbang jalan raya, menjadi lebih lama karena proses bongkar muat memakan waktu lama. Sementara dengan kereta api secara waktu jauh lebih efisien, tetapi dari segi biaya tidak kompetitif dengan pengangkutan melalui jalan raya.
	Untuk jalan raya, misalnya, dipelihara sepenuhnya oleh pemerintah. Sementara rel kereta api pemeliharaannya diserahkan kepada operator. Selain itu, pengguna jasa kereta api dikenai pajak sebesar 10 persen.
	Untuk kondisi saat ini, angkutan barang dengan kereta api memang tidak kompetitif. Namun, kalau aturan mengenai kelebihan muatan benar-benar diterapkan, biaya pengangkutan dengan kereta api seharusnya menjadi lebih murah ketimbang dengan truk, kata Djoko.
	Senin lalu, Gubernur Jateng Ganjar Pranowo mengatakan, langkah pertama yang akan dibenahi adalah persoalan di jembatan timbang agar tidak ada lagi truk dengan kelebihan muatan melalui jalan raya. Selain itu, tidak boleh lagi ada pungutan liar yang menyebabkan daerah tidak mendapat pemasukan berupa denda. Ke depan, diharapkan kelebihan muatan tidak lagi ditoleransi.
	Ganjar sepakat dengan penggunaan moda transportasi lain seperti kereta api untuk angkutan barang. Namun, untuk merealisasikan hal itu, dibutuhkan waktu lebih lama, misalnya untuk menunggu jaringan kereta api memasuki kawasan industri sehingga industri tertarik menggunakan rel kereta api untuk distribusi logistik. Selama menunggu hal itu terealisasi, pembenahan di jalan raya harus dimulai.
	Pungli tak ada lagi
	Sejumlah sopir yang melintasi jalur utama pantura dari arah Jakarta menuju Jateng dan Jawa Timur mengaku tidak lagi dikenai pungutan liar di jembatan timbang di Jateng.
	Hal ini seperti dikatakan Didi (35), sopir truk trailer yang mengangkut besi dari Surabaya menuju Jakarta, saat beristirahat di Plered, Kabupaten Cirebon, Jawa Barat. Didi mengatakan, dirinya tidak lagi dikenai pungli di jembatan timbang Subah di Batang dan jembatan timbang Sarang di Rembang.
	Saya biasanya kena Rp 30.000 di setiap jembatan timbang itu. Namun, sejak berangkat dari Surabaya kemarin, saya tidak kena pungli, kata Didi.
	Namun, sejumlah sopir truk yang terkena tilang di jembatan timbang wilayah Jateng karena muatannya melebihi ketentuan mengeluh karena tetap harus membayar denda kepada petugas jembatan timbang di luar Jateng.
	Sokibri (45), sopir truk trans Jawa, di pangkalan truk Klaling, Kudus, Jateng, mengatakan, para sopir biasa menyebut denda tersebut sebagai biaya kompensasi agar bisa melanjutkan perjalanan. Biaya kompensasi tersebut mulai dari Rp 30.000 hingga Rp 70.000. Pungutan itu terjadi di jembatan timbang wilayah Jawa Timur dan Jawa Barat.
	Seharusnya hal itu tidak perlu terjadi. Kami, kan, sudah menunjukkan surat tilang, kok masih dipungut biaya, kata Sokibri. Padahal, dengan ditilang, sopir harus menanggung biaya tilang sebesar Rp 100.000-Rp 300.000.
